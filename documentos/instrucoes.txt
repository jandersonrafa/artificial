### DIMITRI VIADO ##########
#############################

1) Criar conta em https://github.com/
2) Instalar git https://git-scm.com/
	Comandos basicos:
	git clone: copia todos arquivos do repositorio para a maquina
	git checkout -b nome_branch: cria uma nova branch
	git add: adiciona arquivos para commitar
	git commit -m '': comita localmente os arquivos
	git push origin nome_branch: jogar para o github os teus arquivos alterados localmente
	git checkout nome_branch: troca de branch
	git pull origin nome_branch: carrega alteracoes naquela branch

3) Instalar vscode https://code.visualstudio.com

4) Instalar banco mongo e iniciar serviço
https://stackoverflow.com/questions/20796714/how-do-i-start-mongo-db-from-windows

5) Abrir projeto no vscode 

6) No vscode clicar em debug

7) acessar localhost:8080
